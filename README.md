# ESG_PROJET

**ESG Country** — Analyse des performances ESG des pays

---

## 1. Description du projet

ESG_PROJET est une application / plateforme d’analyse ESG (Environnement, Social, Gouvernance) des pays du monde.  
L’objectif est de fournir des visualisations claires (graphiques, cartes, tableaux) des scores ESG pour chaque pays, afin d’identifier les meilleures performances, les tendances dans le temps, et les domaines à améliorer.

---

## 2. Fonctionnalités

- Import & collecte des données ESG depuis des sources ouvertes (API, jeux de données)  
- Nettoyage et traitement des données (gestion des valeurs manquantes, normalisation)  
- Calcul d’un **score ESG global** et de scores E, S et G par pays  
- Visualisation interactive :  
  - Graphiques (barres, lignes)  
  - Carte du monde colorée selon les scores ESG  
  - Tableaux comparatifs  
- Comparaison entre régions ou groupes de pays  
- Export des résultats (CSV, images)
